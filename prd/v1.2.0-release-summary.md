# v1.2.0 Release Summary

## Status: Ready for Testing & Merge

### Overview
Version 1.2.0 includes 8 completed PRDs focused on metadata handling improvements, UX enhancements, and user preferences. All features have been implemented, tested, and documented.

### Completed Features (8 PRDs)

#### **Critical (P1) - 2 PRDs**
1. **PRD-27: FLAC/OGG Metadata Write**
   - Full Vorbis comment support for FLAC and OGG files
   - Writes title, author, narrator, publisher, language, genre, description
   - 17 new unit tests (all passing)
   - Files: `lib/services/writers/flac_writer.dart`, `lib/services/writers/ogg_writer.dart`

2. **PRD-28: Book List Click Behavior**
   - Clicking book text selects book (switches to detail view)
   - Clicking checkbox toggles batch selection
   - Improves usability for common workflow
   - Files: `lib/main.dart`

#### **High Priority (P2) - 3 PRDs**
3. **PRD-22: Subtitle/OPF Conflict Fix**
   - Changed OPF subtitle to use `<meta name="subtitle">` format
   - Fixed MP4 subtitle atom (©nam → ©st3)
   - Subtitle now survives round-trip correctly
   - 3 new unit tests
   - Files: `lib/services/opf_parser.dart`, `lib/services/writers/mp4_writer.dart`

4. **PRD-23: MP4 Narrator Tag Documentation**
   - Documented that narrator uses ©wrt (composer) atom
   - Added comments explaining OPF is authoritative source
   - Files: `lib/services/writers/mp4_writer.dart`, `lib/services/scanner_service.dart`

5. **PRD-29: UX Improvements (Documented, Not Implemented)**
   - Comprehensive UX audit completed
   - Design system documented in PRD
   - **Deferred to separate feature branch** for safer implementation
   - File: `prd/prd-29-P2-ux-improvements.md`

#### **Medium Priority (P3) - 3 PRDs**
6. **PRD-25: Scan Progress Indicator**
   - Live book count during scan
   - Determinate progress bar (found/total)
   - Better user feedback during long scans
   - Files: `lib/services/scanner_service.dart`, `lib/controllers/library_controller.dart`, `lib/main.dart`

7. **PRD-24: Preferences Persistence**
   - Auto-restore last-opened folder on launch
   - Persist sort order across sessions
   - Save/restore window size and position
   - New service: `lib/services/preferences_service.dart`
   - Files: `lib/main.dart`, `lib/controllers/library_controller.dart`, `pubspec.yaml`

8. **PRD-18: Copy Metadata from Another Book**
   - "Copy from…" button in detail panel
   - Searchable dialog with field selection
   - Copy author, narrator, series, genre, publisher, language
   - New widget: `lib/widgets/copy_from_dialog.dart`
   - Files: `lib/screens/book_detail_screen.dart`, `lib/main.dart`

9. **PRD-16: Rename Folder**
   - "Rename folder" action in overflow menu
   - Proposes name based on current metadata
   - Collision detection and error handling
   - Updates all internal paths automatically
   - Files: `lib/screens/book_detail_screen.dart`, `lib/controllers/library_controller.dart`, `lib/main.dart`
